# Stoic — Daily Wisdom

A mobile-first micro-learning web app for Stoic philosophy. Browse curated quotes, tap to reveal explanations and modern scenarios, then test your understanding with quick comprehension checks.

## Features

- 10 curated Stoic quotes from Marcus Aurelius, Epictetus, and Seneca
- Tap-to-expand explanations with core meaning and modern scenarios
- Multiple-choice comprehension checks with kind feedback
- Light/dark mode with system preference detection
- Swipe navigation on mobile, keyboard navigation on desktop
- Clean, minimalist design optimised for phone screens

## Stack

- Vanilla HTML, CSS, JavaScript
- Zodiak (display) + Satoshi (body) from Fontshare
- Express static server
- No build tools, no dependencies beyond Express

## Run locally

```bash
npm install express
node server.js
# Open http://localhost:5000
```
